# Шредер

Шредер - это программа для безопасного удаления файлов с жесткого диска. В отличие от обычного удаления файлов, которое просто удаляет ссылку на файл в файловой системе, Шредер перезаписывает данные файла случайными байтами несколько раз, что делает восстановление файла практически невозможным.

## Описание

Программа реализует алгоритм безопасного удаления файлов, многократно перезаписывая данные случайными значениями перед удалением. Это предотвращает возможность восстановления удаленных файлов с помощью специализированного программного обеспечения.

## Применение

Программа может применяться в следующих случаях:
- Удаление конфиденциальных документов
- Очистка жесткого диска перед продажей или утилизацией
- Обеспечение конфиденциальности при удалении персональных данных
- Удаление временных файлов, содержащих важную информацию

## Возможные расширения

1. Графический интерфейс пользователя (GUI) на PyQt5/PySide
2. Поддержка различных алгоритмов безопасного удаления (Gutmann, DoD 5220.22-M и др.)
3. Рекурсивное удаление каталогов
4. Интеграция с контекстным меню операционной системы
5. Логирование операций удаления
6. Настройка количества проходов перезаписи
7. Проверка успешности удаления
8. Поддержка перетаскивания файлов (drag and drop)
9. Многопоточная обработка нескольких файлов
10. Локализация интерфейса на разные языки
